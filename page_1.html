Basic Tags in HTML

1. "HTML" tag
It defines the beginning and end of the html document.
Starts with <html>
Ends with </html>

2. "HEAD" tag
It is used for metadata such as title or style for the the html document.
<head>
</head>

3. "TITLE" tag
It is used to give a title to the html document.
<title>
</title>

4. "STYLE" tag
It is used to define style for the html file.
<style>
</style>

The "</head>" tag is used here to end the head of the file.

5. "BODY" tag
It defines the primary content of the document like the paragraphs, images, multimedia, tables, links, lists, etc.
<body>
</body>

6. "HEADER" tag
It is used to give a header to the document. Like the header of the webpage.
<header>
</header>

7. "MAIN" tag
Contains main information related to page's main topic.
<main>
</main>

8. "FOOTER" tag
Used to define a footer for a webpage or document.
<footer>
</footer>
